# Security Policy

感谢你帮助改进 `clip2agent` 的安全性。

## Supported Versions

当前仓库以 `main` 分支作为主要维护分支。若发现安全问题，请默认基于最新代码确认问题是否仍然存在。

## Reporting a Vulnerability

如果你发现了潜在安全问题，请不要直接公开披露细节。建议通过私下渠道联系维护者，并尽量附带以下信息：

- 问题影响范围
- 复现步骤或最小复现样例
- 可能的攻击面或利用条件
- 建议的修复方向（如果有）

在维护者确认并完成修复前，请避免在公开 Issue、PR 或讨论区中披露可直接利用的细节。
